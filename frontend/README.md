# Qualipharma Frontend HTML

## Run local

1. Abrir una terminal en esta carpeta.
2. Ejecutar:

```bash
python -m http.server 5173
```

3. Abrir `http://localhost:5173`

## Setup

- Backend URL: `http://localhost:8000/api/v1`
- Supabase URL: tu proyecto
- Supabase Anon Key: la public anon key

## Login

Si el usuario no existe, usá **Crear cuenta** desde el panel de login.
